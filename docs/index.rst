Specifications
==============

These documents are a reference to Zope Page Templates standards:
Template Attribute Language (TAL), TAL Expression Syntax (TALES), and
Macro Expansion TAL (METAL). It also describes some ZPT-specific
behaviors that are not part of the standards.
